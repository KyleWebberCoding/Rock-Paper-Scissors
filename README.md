# Rock-Paper-Scissors
This is a beginner rock, paper, scissors project completed as part of the Codecademy Backend course.

## What it does
- This will give a random answer to the input from the user.
- e.g. the user chooses rock and the computer will choose a random number which decides if the computer uses rock, paper or scissors.

## Tech used
- JavaScript.
